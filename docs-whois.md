# ya360 whois

Команда, позволяющая производить поиск сущностей по login или alias

## Вывод подсказки

Вывод подсказки с помощью указания ключа **-h**
```
# ya360 whois -h
usage: ya360 whois [-h] name

positional arguments:
  name        Строка поиска

optional arguments:
  -h, --help  show this help message and exit
```
